# Tutorial Menjalankan Docker MySQL & phpMyAdmin

## 1. Persiapan Docker
- Langkah install docker
- Cara cek docker

## 2. Menjalankan docker-compose
- Penjelasan isi docker-compose.yml
- Cara menjalankan: `docker compose up -d`
- Cara stop: `docker compose down`

## 3. Akses MySQL & phpMyAdmin
